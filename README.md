# Proyecto Django CRUD

## Pasos para dejar el proyecto listo

1. Instalar Git, Python y VS Code.
2. En VS Code instalar las extensiones Python y Pylance.
3. Aceptar la invitación del repositorio en GitHub.

## Clonar el proyecto

git clone https://github.com/You9123/Proyecto-Django.git
cd Proyecto-Django
code .

## Crear entorno virtual

python -m venv venv

## Activarlo

PowerShell:
.\venv\Scripts\Activate

CMD:
venv\Scripts\activate

## Instalar dependencias

pip install -r requirements.txt

## Ejecutar el proyecto

python manage.py migrate
python manage.py runserver

## Trabajo en equipo

Cada integrante debe crear su rama:

git checkout -b nombre-de-su-rama
